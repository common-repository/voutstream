== Installation ==

1. Simply using WP Dashboard go to Plugins -> Add New and search for Voutstream Video Ad Manager 
2. Once the plugin has been uploaded, activate the plugin.
3. Once activated, click on the "Settings" link that appears at the top.
4. Under Licence Tab enter Licence ID and Key (empty by default) 
5. Use "Don't have licence key" Link to obtain licence 
6. Register on voutstream.com website and licence will be on the Dashboard 
7. Enter Licence ID and Key in WP and click Save.
